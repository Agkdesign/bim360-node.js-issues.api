Instructions

  1. You need an account on BIM 360 and are member of some projects of Docs
  2. Click the Sign in button to login
  3. A folder tree will appear on the left. Click the item to expand
  4. Click one BIM project, set the due date, all document issues at the due date will be listed
  5. Click the issue to expand its attributes, comments collection and attachments collection
  6. Expand any comment, some comments details will be listed
  7. Click any attachment, the corresponding attachment file will be downloaded to local
  8. If attributes of the issue contains [pushpin], click the pushpin, the corresponding model will be loaded
    9. and the corresponding pushpin markup will be shown up (currently, the demo implementes pushpin in 3D model only
  9. Input issue title, issue due time on the bottom toolbar. Click [Create Issue], one new document issue will be created
    10. and the Issue tree will be refreshed.
  10. Select one issue, input comment body on the bottom toolbar, click [Create Comments], one new comment will be added to this selected issue
    11. and the Issue tree will be refreshed.
  11. Select one issue, choose a local picture, click [Attach Image], one new attachment will be added to this selected issue
    12. and the Issue tree will be refreshed.
  12. Select one BIM 360 project, click [Export CSV], all document issues of this project will be exported to a CSV file.
